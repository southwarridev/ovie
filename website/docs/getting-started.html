What is Ovie?

Ovie is a modern, self-hosted programming language designed to make programming accessible to everyone. Unlike traditional programming languages with cryptic syntax, Ovie uses natural language patterns that are easy to read and understand.

Key Features

  • Self-Hosted: The Ovie compiler is written in Ovie itself
  • Natural Syntax: Uses pidgin English keywords like seeAm instead of print
  • Offline-First: Complete development environment with no network dependencies
  • AI-Friendly: Designed for seamless AI/LLM integration
  • Memory Safe: Built-in memory safety without garbage collection overhead
  • Deterministic: Identical inputs always produce identical outputs

Installation

Getting Ovie installed on your system is quick and easy. Choose the method that works best for you:

Quick Install (Recommended)

üí° One-Line Installation

The fastest way to get started with Ovie.

Unix/Linux/macOS:

curl -sSL https://install.ovie-lang.org | sh

Windows (PowerShell):

iwr -useb https://install.ovie-lang.org/install.ps1 | iex

Build from Source (Offline)

For complete offline installation or if you want to build from source:

git clone https://github.com/southwarridev/ovie.git
cd ovie
make build
make install

Verify Installation

After installation, verify that Ovie is working correctly:

ovie --version

You should see output similar to:

Ovie Programming Language v1.0.0
Self-hosted compiler (written in Ovie)
Build: 2026-01-30

If you see the version information, Ovie is successfully installed and ready to use.

Basic Syntax

Ovie's syntax is designed to be natural and readable. Here are the fundamental concepts:

Variables

// Immutable variables (default)
name = "Alice"
age = 25

// Mutable variables
mut counter = 0
mut score = 100

Functions

// Simple function
fn greet(name) {
    seeAm "Hello, " + name + "!"
}

// Function with return value
fn add(a, b) {
    return a + b
}

// Using functions
greet("World")
mut result = add(5, 3)
seeAm "5 + 3 = " + result

Control Flow

// Conditionals
mut age = 18

if age >= 18 {
    seeAm "You are an adult"
} else {
    seeAm "You are a minor"
}

// Loops
for i in 0..5 {
    seeAm "Count: " + i
}

// While loops
mut count = 0
while count < 3 {
    seeAm "Count: " + count
    count = count + 1
}

Data Structures

// Structs
struct Person {
    name: String,
    age: Number,
    email: String,
}

// Creating and using structs
mut person = Person {
    name: "Alice Johnson",
    age: 30,
    email: "alice@example.com",
}

seeAm "Name: " + person.name
seeAm "Age: " + person.age

Arrays and Collections

// Arrays
mut numbers = [1, 2, 3, 4, 5]
mut names = ["Alice", "Bob", "Charlie"]

// Accessing elements
seeAm "First number: " + array_get(numbers, 0)
seeAm "Array length: " + array_length(numbers)

// Adding elements
append(numbers, 6)
seeAm "New length: " + array_length(numbers)

üí° Natural Language Keywords

Notice how Ovie uses natural language patterns:

  • seeAm instead of print or console.log
  • mut for mutable variables (clear and concise)
  • array_get and array_length for clear intent
