üîß Troubleshooting

Windows Users:
  • Use PowerShell for .ps1 installer or Command Prompt for .bat installer
  • Install Git from git-scm.com if needed
  • Ensure Rust toolchain is installed for building from source
Linux/macOS Users:
  • Ensure curl or wget is installed
  • Install Rust toolchain: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Make sure build tools are available (gcc, make, etc.)
Need Help?
  • Report issues on GitHub
  • Ask questions in Discussions
  • Check Installation Guide for detailed instructions
