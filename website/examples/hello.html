Hello World Example

Your first program in the self-hosted Ovie language

Difficulty: Beginner
Category: Basic
Concepts: Output, Variables, Functions
Compiler: Self-Hosted Ovie

// Hello World in Ovie - Self-Hosted Programming Language!
// This code is compiled by a compiler written in Ovie itself

seeAm "Hello, World!"
seeAm "Welcome to Ovie programming!"

// Let's explore some basic features
mut name = "Developer"
mut language = "Ovie"
mut year = 2026

// String concatenation
seeAm "Hello, " + name + "!"
seeAm "You're learning " + language + " in " + year

// Function definition
fn greet(person) {
    seeAm "Greetings, " + person + "!"
    seeAm "Welcome to the self-hosted programming language!"
}

// Function call
greet("New Programmer")

// Simple calculation
mut a = 10
mut b = 5
mut sum = a + b

seeAm "Calculation: " + a + " + " + b + " = " + sum

// Conditional logic
if sum > 10 {
    seeAm "The sum is greater than 10!"
} else {
    seeAm "The sum is 10 or less."
}

üîç Code Explanation

This Hello World example demonstrates several fundamental concepts in Ovie:

1. Natural Language Output

Ovie uses seeAm instead of traditional print statements. This makes the code more readable and accessible to non-programmers.

2. Variables and Mutability

Variables are declared with mut for mutable variables. Ovie encourages explicit mutability for better code safety.

3. String Concatenation

Strings are combined using the + operator, making it intuitive for beginners.

4. Functions

Functions are defined with fn and can take parameters. The syntax is clean and readable.

5. Conditional Logic

If-else statements work as expected, with clear syntax that reads like natural language.

Ready to run this code? Follow these steps:

  1. Create a new Ovie project: ovie new hello-world

  3. Copy the code above into src/main.ov
  4. Run the program: ovie run
